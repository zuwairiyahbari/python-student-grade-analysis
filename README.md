# Python Student Grade Analysis
A Python application that analyzes student performance using pandas and SQLite.

## Features

- Reads student assessment data from a CSV file
- Calculates weighted final grades
- Assigns letter grades automatically
- Stores processed records in a SQLite database
- Sorts students alphabetically using SQL queries
- Generates a frequency distribution of letter grades
- Removes failing grades from the database

## Technologies

- Python
- pandas
- SQLite
- SQL

## Files

- `Ten_Students_Scores.csv` — Input dataset containing student assessment scores.
- `Ten_Students_Grades.csv` — Generated output containing calculated numeric and letter grades.
- `Student.db` — SQLite database generated when the program runs.
